# Eye Drop Explorer

## 如何開啟

直接用瀏覽器開啟 `eye_drop_explorer.html` 即可使用。資料已嵌入 HTML，不需要網路或伺服器。

## 使用資料

- `eye_meds_tw_raw.csv`
- `eye_meds_tw.xlsx`
- `verification_report_eye_meds.md`
- `excluded_eye_products.csv` 僅作為資料篩選背景參考

## 功能

- 依商品、成分、適應症、用途分類查詢
- 支援搜尋、下拉選單、藥品分類、防腐劑、劑型與用途分類篩選
- 顯示代表產品清單，可展開查看完整適應症、完整成分、許可證持有廠商、備註與資料來源
- 內建成分/適應症/產品數排行、藥品分類比例、防腐劑統計圖

## 限制

- 本工具只反映既有資料庫中的許可證、成分與適應症資訊，未重新抓取 TFDA 資料。
- 原資料庫中藥品分類多數因官方欄位空白而標示為「待人工確認」。
- 成分、用途與適應症關聯為資料整理結果，實際用藥仍需由藥師或醫師依個案判斷。
- 未使用外部 CDN 或第三方前端套件。
