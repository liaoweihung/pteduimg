# 網站更新紀錄

這份紀錄用來整理藥局衛教助手的功能更新、圖卡新增與維護修正。  
目前內容先依據 2026-03-24 以後的 Git 提交紀錄回填；部分早期提交訊息較簡略，例如 `Add files via upload`，因此只保留能從檔案與卡片資料確認的重點。

## 維護方式

之後每次新增功能或圖卡時，建議在最上方追加一筆：

```text
## YYYY-MM-DD

### 新增圖卡
- 圖卡系列名稱：新增 N 張圖卡。

### 功能更新
- 功能名稱：簡短說明。

### 修正與維護
- 修正項目：簡短說明。
```
